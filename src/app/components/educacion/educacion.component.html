    (2021-Actualmente)

  • Desarrollador Full Stack

    -

    (1 year)

    • Cuento con un perfil de profesional versátil capaz de desarrollar tanto el frontend como el backend de una aplicación web. Poseo conocimientos en múltiples tecnologías y sus aplicaciones, desde la interfaz de usuario hasta la lógica de negocio y la base de datos.

  • QA tester (Testing manual & Testing automation)

    -

    (1 year)

    • El papel que desempeño como QA Tester es asegurar la calidad y funcionalidad de un software o sistema determinado, donde se debe lograr identificar y resolver problemas para asegurarle al usuario una experiencia optima y sin errores.
